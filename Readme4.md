# Password Generator

## Description
This is a simple Python-based Password Generator that creates secure and random passwords based on user preferences. Users can choose whether to include letters, numbers, and special characters, specify the password length, and generate multiple passwords in one session.

## Features
- Generate random passwords
- Select password length
- Include letters, numbers, and symbols
- Display password strength (Weak, Medium, Strong)
- Generate multiple passwords without restarting the program
- Easy-to-use command-line interface

## Technologies Used
- Python 3
- random module
- string module

## How to Run
1. Make sure Python 3 is installed.
2. Save the program as `password_generator.py`.
3. Open a terminal or command prompt.
4. Run the following command:

```bash
python password_generator.py
```
