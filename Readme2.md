# BMI Calculator using Python

## Description
This is a simple Python-based BMI (Body Mass Index) Calculator that calculates a user's BMI based on their weight and height. The program also classifies the BMI into different health categories and allows users to perform multiple calculations in a single session.

## Features
- Calculate Body Mass Index (BMI)
- Classify BMI into:
  - Underweight
  - Normal Weight
  - Overweight
  - Obese
- Input validation for positive weight and height values
- Repeat calculations without restarting the program
- User-friendly command-line interface

## Technologies Used
- Python 3

## Concepts Used
- Functions
- While Loop
- Conditional Statements (`if`, `elif`, `else`)
- User Input
- Arithmetic Operations

## How to Run
1. Install Python 3 on your system.
2. Save the program as `bmi_calculator.py`.
3. Open a terminal or command prompt.
4. Run the following command:
   ```
   python bmi_calculator.py
   ```
5. Enter your weight (kg) and height (m) when prompted.
