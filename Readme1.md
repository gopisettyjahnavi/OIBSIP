#  Python Voice Assistant

A simple voice-controlled assistant developed using Python that listens to user commands and performs basic tasks. This project demonstrates the use of speech recognition, text-to-speech, and browser automation, making it a great beginner-level automation project.

## Features
-  Recognizes voice commands
-  Responds using text-to-speech
-  Tells the current time
-  Opens Google
-  Opens YouTube
-  Greets the user
-  Exits the application using a voice command

## Technologies Used
- Python
- SpeechRecognition
- pyttsx3
- Webbrowser
- Datetime

## Required Libraries
pip install SpeechRecognition
pip install pyttsx3
